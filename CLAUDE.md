# Project: Czech Race Aggregator

## Design

Follow the Scandinavian minimalist design system defined in [DESIGN.md](./DESIGN.md). All UI work must adhere to the color palette, typography, spacing, and component patterns described there. When in doubt, choose the simpler, quieter option.

Key principles:
- Warm off-white background (`#F5F3EF`), white cards, deep green (`#2D4A3E`) accents
- Generous whitespace, `16px` rounded cards, subtle shadows
- Inter font, clear typographic hierarchy
- Lucide Icons (outlined, 1.5px stroke)
